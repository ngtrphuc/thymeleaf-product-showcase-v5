# Screenshots Checklist

Use this checklist to capture portfolio-ready media for README and project showcase.

## Storefront

- `storefront-products.png` (catalog grid + filters)
- `storefront-product-detail.png` (detail + recommended section)
- `storefront-cart.png` (cart summary with totals)
- `storefront-checkout.png` (checkout information and payment method)
- `storefront-orders.png` (order history)

## Admin

- `admin-dashboard.png` (KPIs + recent orders)
- `admin-products.png` (product management list/search)
- `admin-orders.png` (order state management)
- `admin-chat.png` (real-time support chat panel)

## Monitoring

- `monitoring-overview.png` (Grafana: Smartphone Shop Overview dashboard)
- `monitoring-alerts.png` (Prometheus active alerts page)

## Capture Guidelines

- Recommended size: `1440x900` or higher.
- Use browser zoom `100%`.
- Hide personal browser extensions/tabs.
- Prefer realistic seed data over empty states.
- If using GIF, keep each clip under 15 seconds and name as `*.gif`.
